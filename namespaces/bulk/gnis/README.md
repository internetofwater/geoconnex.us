# GNIS Bulk Integration

The Geographic Names Information System (GNIS) is the U.S. federal standard for geographic nomenclature, developed by the U.S. Geological Survey (USGS) in cooperation with the U.S. Board on Geographic Names (BGN). It acts as the official repository for over 2 million physical and cultural features—such as cities, lakes, mountains, and cemeteries—across the United States and its territories.

This is a Geoconnex bulk integration due to the fact GNIS is a very large dataset. As such, a crawler client for Geoconnex must run the docker image specified in the [sitemap xml](./gnis.xml)
