# NHD and WBD dataset relationships
**NHDPlusV2** – Built on Medium resolution NHD data using older WBD snapshots, and the 30m National Elevation Dataset. Alaska is not included in this dataset; 4-digit HUCs create the outer extent of the data within CONUS. For more info: https://www.epa.gov/waterdata/get-nhdplus-national-hydrography-dataset-plus-data  

**NHDPlusHR** – Built on NHDPlusV2, it improves on previous work with increased spatial resolution of 10m 3D Elevation Product (3DEP) DEM data. Other additions include full 8-digit HUCs for international basins across the border of Mexico and Canada, and portions of Alaska. For more info: https://www.usgs.gov/national-hydrography/access-national-hydrography-products 


**WBD2020** – This dataset was used as the basis for USGS national water availability assessments and remains linked to that assessment. It predates the full 2012-2024 national review, is missing a few units near the Great Lakes, and has less Alaskan coverage than WBD 2025. For more detailed differences between WBD 2020 and WBD 2025, see: https://water.usgs.gov/themes/hydrologic-units/#nhdplus-v2-wbd-snapshot 

**WBD2025** – This is the current official version from USGS WBD, which incorporates updates from previous releases including partner edits which improved attribution, geometry and topology. It includes coverage for all U.S. islands and territories, and an expansion for Alaskan areas. For more detailed differences between WBD 2020 and WBD 2025, see: https://water.usgs.gov/themes/hydrologic-units/#nhdplus-v2-wbd-snapshot
