OGC(r) Web Coverage Implementation Schema Standard (CIS)
-----------------------------------------------------------------------

This Coverage Implementation Schema (CIS) specifies the OGC coverage
model by establishing a concrete, interoperable, conformance-testable
coverage structure.

More information may be found at
 http://www.ogc.org/standards/wcs

The most current schema are available at http://schemas.opengis.net/cis/
and all OGC schemas may be downloaded in a complete bundle from
http://schemas.opengis.net/SCHEMAS_OPENGIS_NET.zip
