OGC(r) Earth Observation SPS Extension 2.0 Schema
================================================================

OGC(r) Sensor Planning Service Interface Standard 2.0
 Earth Observation Satellite Tasking Extension 2.0
-----------------------------------------------------------------------

The OGC Sensor Planning Service Interface Standard 2.0 
Earth Observation Satellite Tasking Extension 2.0 is defined in 
OGC document 10-135.

More information may be found at
 http://www.opengeospatial.org/standards/sps

The most current schema are available at http://schemas.opengis.net/
and all OGC schemas may be downloaded in a complete bundle from
http://schemas.opengis.net/SCHEMAS_OPENGIS_NET.zip

The root (all-components) XML Schema Document, which includes
directly and indirectly all the XML Schema Documents, defined by
the standard is eosps.xsd.

* Latest version is: http://schemas.opengis.net/eosps/2.0/eosps.xsd *
