OGC(r) Moving Features Encoding Part I: XML Core Standard
-----------------------------------------------------------------------

The OGC(r) Moving Features Encoding Part I: XML Core Standard specifies
standard encoding representations of movement of geographic features. The
primary use case is information exchange.

More information may be found at
 http://www.opengeospatial.org/standards/movingfeatures

The most current schema are available at http://schemas.opengis.net/ 
and all OGC schemas may be downloaded in a complete bundle from
http://schemas.opengis.net/SCHEMAS_OPENGIS_NET.zip

* Latest version is: http://schemas.opengis.net/movingfeatures/1.0/mf-xmlcore.xsd *
